════════════════════════════════════════════════════════════════
VISUAL VOCABULARY
════════════════════════════════════════════════════════════════

LONG-SIDE ELEVATION:
  Thick vertical rectangle         → POTEAU
  Thick horizontal at top          → SABLIERE
  Thick intermediate horizontal    → PANNE
  Single diagonal in panel         → PALÉE DE STABILITÉ
  Two diagonals crossing (X)       → CONTREVENTEMENT CVT
  Element with beveled end         → JARRET

ROOF PLAN:
  Large longitudinal beams         → TRAVERSE
  Diagonals in horizontal plane    → POUTRE AU VENT
  Short corner diagonals           → DRETELLES
  Thin perpendicular bars          → LIERNE (round bar D14 typ.)
  Regular parallel grid            → PANNES COURANTES
  Central ridge beam               → PANNE FAÎTIÈRE
  Low perimeter beam               → SABLIERE

GABLE ELEVATION:
  Main vertical façade columns     → POTEAU PIGNON
  Small intermediate columns       → POTELET
  Dense horizontal/vertical grid   → LISSES + MONTANTS BARDAGE

ASSEMBLY DETAILS:
  Flat rectangle + thickness cote  → PLATINE (PL) or Tôle Noire (TN)
  Triangular/trapezoidal plate     → GOUSSET
  Small angle cleat fixing purlin  → ÉCHANTIGNOLE
  Threaded rod noted M20/M24/ø20   → TIGE D'ANCRAGE
  Vertical plate inside web        → RAIDISSEUR

DO NOT EXTRACT:
  ─ ─ ─   dimension/axis lines
  ←────→  dimension lines with arrows
  ○  ⊕    bolt/weld symbols
  B1, P1  reference tags (note them, never extract as profiles)
  NERVESCO / sheeting / colliers / visserie

LENGTH RULE — ZERO AMBIGUITY
════════════════════════════════════════════════════════════════

CAS 1 — Explicit cote on element or dimension line:
  → length_m      = cote in metres (4000mm → 4.0)
  → length_source = "explicit_dimension"

CAS 2 — Building span visible but NO explicit cote on the piece:
  → PANNE, TRAVERSE, SABLIÈRE, LISSE, LIERNE:
    MUST use building span as cut length
    → length_m      = span in metres
    → length_source = "span_used_as_cut_length"
  → All other elements:
    → length_m      = null
    → length_source = "span_only_not_applicable"

CAS 3 — No dimension visible:
  → length_m      = null
  → length_source = "no_dimension_visible"

ABSOLUTE: length_m always in METRES.
ABSOLUTE: never invent a length — null beats fabrication.

ACCESSORIES FORMAT RULES (platines, goussets, échantignolles)
════════════════════════════════════════════════════════════════

PLATINES / TÔLES NOIRES:
  → MANDATORY format: "TN ep*larg*long" or "PL ep*larg*long"
  → All three dimensions REQUIRED
  → Valid:   "TN 20*100*250" / "PL 25*450*450"
  → INVALID: "PL25" / "TN 20" / "Platine EP25"
  → category = "platine" / length_m = null

GOUSSETS:
  → MANDATORY format: "TN ep*larg*long"
  → Valid:   "TN 8*220*180"
  → INVALID: "Gousset EP8"
  → category = "gousset" / length_m = null

ÉCHANTIGNOLLES:
  → MANDATORY format: "TN ep*larg*long"
  → Valid:   "TN 8*70*80"
  → INVALID: "Echantignole EP8"
  → category = "echantignole" / length_m = null

TIGES D'ANCRAGE:
  → Format: "4ø20 L=600" / "Canne Ø20 L=500"
  → length_m = explicit length (600mm → 0.6), else null
  → category = "tige_ancrage"

RAIDISSEURS:
  → MANDATORY format: "TN ep*h*e" or "PL ep*larg*long"
  → category = "raidisseur" / length_m = null

3. LENGTH UNITS IN TABLES: Lengths in tables are often in millimeters (e.g., 1340). You MUST convert them to metres (e.g., 1.34). If the length column is blank or "----", set length_m to null.

COMPUTATION RULES
════════════════════════════════════════════════════════════════
Linear profiles (Standard):
  poids_total_kg = poids_lineaire_kg_m × length_m × quantity

FER PLATS (PL, FER PLAT):
  Identification: PL150x6, PL 150x6, PL150*6, FERPLAT 150x6, FER PLAT 150x6 (150=width mm, 6=thickness mm)
  Priority 1: Use catalogue weight.
  Priority 2: poids_lineaire_kg_m = larg_mm × ep_mm × 0.00785
  poids_total_kg = quantity × length_m × poids_lineaire_kg_m

Plates (Platines PLT/TN, goussets, échantignolles, raidisseurs):
  Format: PLT300x250x10 or TN 10*250*300
  poids_total_kg = (long_mm/1000) × (larg_mm/1000) × (ep_mm/1000) × 7850 × quantity

summary.total_weight_kg = sum of all non-null poids_total_kg
summary.extraction_confidence_avg = mean of all confidence values
